Releasio Minecraft

For Minecraft Bedrock Edition (Java version coming soon)

Standard Version 

All blocks and items are BUP, but some are changed, such as ores and glass. Colored wool, glass, and Clay are colored accordingly, and leaves and grass are green. 


Default Items Version  

Same as standard version, except the items are the original Minecraft textures.


No Music Version   

Same as default version, but with no custom music.


Default Items and No Music Version  

Has no custom music, and item textures are the original Minecraft textures. 


EXTREME VERSION

All textures are Toad's face without any changes made. Colored blocks are not colored, grass and leaves are not green, and every line of text in the game is replaced with "BUP".
